# Project Instructions

This is a functional prototype for a B2B conversational quoting platform.

## Stack
- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- lucide-react
- Local mock data only

## Product principles
- Chat-first experience
- AI is the commercial brain
- Dynamic UI adapts to the conversation
- Quote builder is always visible
- WhatsApp-to-web continuity is central
- No generic chatbot widget
- No traditional e-commerce layout

## Design style
- Premium B2B SaaS
- Clean, modern, precise
- Rounded cards
- Soft shadows
- Neutral backgrounds
- Blue/teal accents
- Data-dense but readable

## Code standards
- Reusable components
- No large components over 250 lines unless justified
- Business logic goes in /lib
- Mock data goes in /data
- UI components go in /components
- Use TypeScript types clearly
- Run lint/build after implementation
